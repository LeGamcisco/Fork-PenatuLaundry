# Penatu

Laundry Management App

## Fitur 🚩

- Showcase
- Login / Register
- Dashboard
- Order Histori
- Profile
- Stats (Today, month order)
- Pesanan


## Mockup 🎨
- [Penatu](/asset/.png 'Penatu')

## Instalasi 🔧

1. clone repository
2. flutter pub get
3. create .env & isi confignya
4. flutter run
